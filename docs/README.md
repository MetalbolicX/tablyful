# tablyful

## What problem tablyful solves?

Tablyful is a TypeScript library designed to convert semi-structured data into common table formats such as CSV, JSON, and Markdown. It simplifies the process of transforming data from various input formats into structured tabular representations, making it easier for developers to handle data manipulation and export tasks.

The Tablyful library is particularly useful for applications of data science or data engineering that require data reporting, exporting, or visualization, as it provides a straightforward API for converting data into widely-used formats. It is inspired by the tabulate package in Python and aims to bring similar functionality to the JavaScript ecosystem.

## Key Features

- ✅ Support for multiple input data formats (arrays, objects, etc.)
- ✅ Conversion to popular table formats (CSV, JSON, Markdown)
- ✅ Customizable formatting options
- ✅ Streaming API for large datasets
